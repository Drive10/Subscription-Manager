# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability, please email the maintainers directly. Do **not** open a public GitHub issue.

You should receive a response within 48 hours. If the issue is confirmed, a fix will be released as soon as possible.
